# Quiz Maker - Valorant Themed

## Created by: Edward McGrath

---

## What This App Does

This app allows you to do a quiz based on any questions you need to answer, no matter how many. These are multiple choice questions, meaning you need to provide four different answers and the correct answer inside the google sheet.

## How to Use It

Create the google sheet
This code runs off of a google sheet, and changing the answers in the google sheet will change the answers and questions provided by the code
Run the Code
Running the code will bring up the quiz, where you just have to answer the questions all the way through to get your “rank”
Change the questions or reset the quiz
The quiz can be reset via one button, and you can change the questions inside the quiz via the google sheet.


---

## Features

- Create your own quiz to study
- Immediate responses, it tells you if you’re wrong and shows the correct answer
- Progress bar to track how far along you are
- A reset button to restart the quiz by just clicking it
- The questions will show up in a random order, with a random placement for each answer

---

## React Concepts Used

- `useState` to manage the quiz’s state, such as score and current question
- ‘useEffect’ helps to fetch the data from the google sheet
- ‘onClick’ to select the buttons and answers provided
- Tailwind for the design
